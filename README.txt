Protótipo do site do casamento Ivana & Gabriel

Arquivos principais:
- index.html: página inicial/apresentação
- confirmacao.html: página de RSVP visual, ainda sem backend
- como-chegar.html: hospedagem e deslocamento
- eventos.html: programação opcional do fim de semana
- casamento.css: paleta e estilos

Para usar:
1. Coloque todos os arquivos na mesma pasta.
2. Abra index.html no navegador.
3. No site final, conecte o formulário de confirmação ao backend ou substitua o botão por um link real.
4. Atualize links de WhatsApp/site/hospedagem quando desejar.
